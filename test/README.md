# Develop4Docker

开发测试环境Docker版，包括nginx，php环境，数据库，redis，memcached，自带phpMyAdmin

> 需要docker和docker-compose

数据库用户名密码：root root

web目录`wwwroot`

访问地址 http://localhost

phpMyAdmin 地址 http://localhost:8080

代码内服务域名

- redis
- memcached
- mysql


> `docker_mirror.sh`为添加 docker hub 阿里云镜像源脚本，可加速镜像拉取
